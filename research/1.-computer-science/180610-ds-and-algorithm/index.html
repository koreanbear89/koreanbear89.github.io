3. Time Complexity

  • n<1,000,000, the expected time complexity is $O(n)$ or $O(nlogn)$

  • n<10,000, the expected time complexity is $O(n^2)$

  • n<500, the expected time complexity is $O(n^3)$




5. Prefix sums

  • There is a simple yet powerful technique that allows for the fast calculation of sums of elements in given slice.

  • Its main idea uses prefix sums which are defined as the consecutive totals of the first 0,1,2, … n elements of an array

$$p_n = a_0 + a_1 + … a_{n-1}$$

  • 주어진 array에 대하여 $p_n$ 으로 이루어진 리스트 $P$를 계산하면 편한 경우가 많음




8. Leader

  • 주어진 sequence $a_0, a_1, … a_{n-1}$ 에서 $\frac{n}{2}$ 이상 나온 원소를 leader라고 함.

  • 8.1 Solution with $O(n^2)$ : 냅다 for문 돌리면 $O(n^2)$

  • 8.2 Solution with $O(nlogn)$ : sort먼저하고 for문 돌리면 $O(nlogn)$

  • 8.1 Solution with $O(n)$ : 만약 어떤 sequence가 leader를 갖고 있다면 a pair of elements of different values 하여도 여전히 그 seq의 leader는 같다.

def goldenLeader(A):
    '''주어진 seq A의 원소를 하나씩 stack 으로 옮기면서 stack의 가장 top에 있는 두 원소가 다르면 둘다 지워버림.
    모든 원소에 대하여 이 작업을 다하고나면 stack에는 leader만 남게됨, O(n)으로 leader 찾기
    # 단 예외처리가 안되어있는것 같으므로 긁어서 사용시 주의요함
    '''

    n =  len(A)
    size, candidate = 0, -1

    for k in range(n):
        if size == 0:
            size += 1
            value = A[k] # 현재 stack 끝에 남은 value라고 생각하면 될 듯
        else:
            if value != A[k]: size -= 1
            else: size += 1

    # leader가 없는경우 stack에 아무것도 남지 않을수도 있음
    if size > 0: candidate = value
    leader = -1
    count = 0

    # stack에 남아는 있는데 2/n개가 안넘어 leader가 아닐수도 있으므로 이를 확인함
    for k in range(n):
        if (A[k] == candidate): count += 1
        if (count > n // 2): leader = candidate

    return leader




9. Maximum slice problem

  • 주어진 seq에서 가장 큰 합을 갖는 slice를 찾는 문제

  • 9.3 Solution with O(n) time complexity : 각 위치에서 끝나는 largest sum을 계산함

def max_slice(A):
    # max_ending은 현재 위치에서 가장 큰 값, max_slice는 현재까지중 가장 큰 값을 저장
    # 다만 모든 값이 음수인 경우 예외처리가 안되어있어 0이 출력됨
    max_ending, max_slice = 0,0
    for a in A:
        max_ending = max(0,max_ending + a)
        max_slice = max(max_slice, max_ending)

    # 모든 값이 음수거나 0인 경우
    if max_slice ==0 : return max(A)

    return max_slice
