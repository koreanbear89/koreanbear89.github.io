1. Stats

  • EPS (Earning Per Share, 순이익/총발행주식): 주당순이익, 즉 기업의 순이익을 발행주식수로 나눈 값

  • PER (Price Earning Ratio, 시가총액/순이익) : 주가수익률, 즉 이 기업을 시총주고 사서 몇년 들고있어야 원금회수가 가능할 것인가.

  • BPS (Bookvalue Per Share, 기업순자삭/총발행주식) : 주당순자산가치

  • PBR (Price Bookvalue Ratio, 현재주가/BPS) : 주가순자산비율, 즉 현재 기업의 주가가 기업의 순자산 가치에 비해 몇배 비싼지

  • ROE (Return On Equity) : 자기자본 이익률

  • ROA (Return On Assets) : 총자산이익률

  • Beta : 변동성지수, 일반적으로 1 이상이면 변동성이 큰 것으로 본다.

  • YoY (Year on Year) : 전년대비

  • TTM (Trailing Twelve Months) : 최근 12개월

  • MRQ (Most Recent Quarter) : 최근 분기

  • 자산 : 유동자산(현금, 채권, 재고) + 비유동자산

  • 자본 : 자산총계 - 부채총계

2. 업종

  • IPO(Initial Public Offering) : 상장, 기업공개, 기업 설립 후 처음으로 외부투자자에게 주식을 공개하고, 이를 매도하는 업무를 의미

  • REITs : 다수의 투자자로부터 자금을 모집하여 부동산에 투자,운용하여 발생하는 수익을 투자자들에게 배당하는 회사

  • BDC(Buisness Development Company, 사업개발회사) : 공모를 통해 모집한 자금으로 중소기업에 주로 투자하는 투자전문회사

  • 목표주가 : PER X EPS
