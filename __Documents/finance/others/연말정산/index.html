연말정산 과정

  • 아래 과정에 따라 기타공제 와 특별세액공제 만 신경쓰면 됨, 다른 부분은 내가 어떻게 할 수 없음

[ + ] 총급여 : 연봉 (급여+상여+수당+..) - 비과세소득

​ [ - ] 근로소득공제 : 근로자의 경우 법정금액을 필요경비로 인정(1200만원 + 4500만원 초과금의 5%)

​ [ - ] 인적공제 : 본인, 배우자, 부양가족 등

​ [ - ] 연금보험 : 국민연금, 공무원, 군인, 사학연금등

​ [ - ] 특별공제 : 건강보험, 고용보험, 주택자금, 기부금

​ [ = ] 차감소득금액

​ [ - ] 기타공제 : 개인연금저축, 주택마련저축, 신용카드

​ [ = ] 종합소득과세표준 : 이 금액에 세율을 곱하고 누진공제액 (108만원) 을 제하면

​ [ = ] 산출세액

​ [ - ] 세액공제 : 근로세액공제 ( 근로소득공제와 같은 맥락)

​ [ - ] 특별세액공제 : 보험료, 의료비, 교육비, 기부금

​ [ = ] 결정세액




기타 (소득) 공제

  • 개인연금저축 + 퇴직연금 : 연 700만원까지 => 700만원의 16.5%인 115만원까지 연말정산에서 돌려받음 : 주택구입시 중도인출은 가능하나 기타소득세 16.5%를 뱉어야하고 개인납입이 아닌 퇴직금의 경우 퇴직소득세를 뱉어야함
  • 주택청약저축 : 연 240만원, 무주택 세대주여야 의미있음
