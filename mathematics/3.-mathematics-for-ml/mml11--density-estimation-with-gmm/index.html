11. Density Estimation with Gaussian Mixture Models

  • Introduction : We often aim to represent data in some way.
    • In density estimation, we represent the data compactly using a density from a parametric family, (e.g., a Gaussian or Beta distrib)
  • In this chapter, we will focus on Gaussian Mixture Models, where the basic distributions are Gaussians.
    • For a given dataset, we aim to maximize the likelhood of the model params to train the GMM

11.1 Gaussian Mixture Model

  • Gaussian Mixture Model : a density model where we combine a fininte number of $K$ Gaussian Distrib.

$$ p(x|\theta) = \sum_k{\pi_k N(x|\mu_k, \Sigma_k)} $$

  • An illustration is given in Figure 11.2, displaying the weighted components and the mixture densitiy, which is given as

$$ p(x|θ)=0.5 N (x|−2,1) + 0.2N(x|1,2) + 0.3N(x|4,1) $$

Figure 11.2 Gaussian Mixture Model composed of a convex combination of Gaussian distrib

11.2 Parameter Learning via Maxinum Likelihood

  • Objective : find a good representation/approximation of given data by means of GMM with $K$ mixture components

    • Given a dataset $X = [x_1, …, x_N ]$

    • parameters of the GMM $\theta$ : means $\mu_k$, covariances $\Sigma_k$, mixture weights $\pi_k$

    • we detail how to obtain a maximum likelihood estimate $θ_{ML}$ of the model parameters $θ$ in the following.

      • $\theta_{ML}$ maximize the log-likelihood $L$

  • Likelihood : of the training data $[x_1, …, x_N ]$ given the parameters

    $$ p(X|\theta) = \prod_n^N p(x_n | \theta), \text{ } p(x_n|\theta) = \sum_k^K \pi_kN(x_n|\mu_k, \Sigma_k) $$

  • Log-Likelihood $L$: every individual likelihood term $p(x_n|\theta)$ is a Gaussian Mixture Density, Then we obtain the log-likelihood as

$$ log p(X|\theta) = \sum_n^N log p (x_n | \theta) = \sum_n^N log \sum_k^K \pi_k N (x_n | \mu_k, \Sigma_k) $$

  • Maximization : compute the gradient $dL/dθ$ of the log-likelihood with respect to the model parameters $θ$, set it to $0$, and solve for $θ$.

    • Unlike previous examples for MLE, we cannot obtain a closed-form solution.

    • We can exploit an iterative scheme to find good params (EM algorithm)
