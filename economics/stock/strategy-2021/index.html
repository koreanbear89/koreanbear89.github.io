  2021-04-01   Strategy  Comments

Strategy.

  • 통화정책에 대한 시장 컨센서스 : 테이퍼링발표(3Q21) -> 테이퍼링진행(1Q22) -> 금리인상(1Q23)

    • 유동성으로 인한 프리미엄이 붙은 자산은 위험 (신흥국, space, 코인)
    • 6~7월 발표되는 물가고용데이터가 좋을 것 : 테이퍼링 발표전에 주가조정이 선행될 것

  • 금리인상 주의업종

    • 재무구조 불건전기업 => 재무구조가 안좋으면 금리인상에 무조건 불리
    • 적자소형성장주 => 금리가 인상되면 생존에 어려움
    • REITs => 금리가 올라가면 리츠의 월세가 갖는 매력도가 떨어짐,
    • 고배당주 => 생각보다 금리가 높으면 이 배당의 매력이 떨어짐

  • 미국 소비자물가지수 (CPI) 체크 : 인플레이션 현황파악

    • 연준은 이번 인플레가 지속가능하지 않다고 보고있으며 6,7월이 피크 올해말에 가라앉기 시작할것이라 보고있음.
    • 따라서 올해말에도 물가상승이 지속된다면 FED의 포지션이 바뀔 것
    • 4월 (4.2%) - 5월 (5.0%) - 6월 (5.4%)

  • 조정, 박스가 길었던 종목들 모아갈것

    • 코로나 특수가 끝나서 재미가 없어진 주식 : AMZN, PYPL
    • 성장성 높으나 조정, 박스구간이 길었던 주식 : DIS ,TSLA
    • 성장성 높으나 지금 고점 : GOOGL, MSFT, QLD
