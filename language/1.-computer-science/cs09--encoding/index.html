Base64

  • Base64 : An encoding method that converts an arbitrary byte stream into ASCII characters that can be displayed on the screen.

  • Consists of :

    • Uppercase alphabet : 26 letters (maps 0-25 for each )
    • Lower case alphabet : 26 letters (maps 26-51 for each )
    • [0-9] , +, / : 12 letters (maps 52-63 for each )
    • “=” : finish code
